# DigitalStop

Built for the Future

## Project Overview

DigitalStop is an online shop focused on selling smartphones and mobile technology. The platform provides users with a simple and efficient way to browse, compare, and purchase modern devices.

The project is designed with scalability and performance in mind, aiming to deliver a smooth user experience across all devices.

## Features

- Wide selection of smartphones  
- Product search and filtering  
- Shopping cart functionality  
- Responsive design  
- Fast and optimized interface

## Technologies Used

- Frontend: HTML, CSS, JavaScript, Angular
- Version Control: Git and GitHub

The goal of DigitalStop is to build a modern and reliable e-commerce platform for smartphones that is easy to use, scalable, and ready for future expansion.
